# React Redux Starter Kit

## Requirements:-

### `node > ^6.0.0`
### `yarn > ^0.23.0 or npm > ^3.0.0`

## Running the Project

### `npm install`
### `npm start`

Open [http://localhost:8081](http://localhost:8081) to view it in the browser.
